# Carrosserie Mounier

Site vitrine premium pour une carrosserie nouvelle génération.

## Mise en ligne GitHub Pages

1. Créer un dépôt `carrosserie-mounier`
2. Ajouter tous les fichiers du dossier
3. Aller dans `Settings > Pages`
4. Choisir `Deploy from a branch`
5. Branch : `main`
6. Folder : `/root`
7. Enregistrer
